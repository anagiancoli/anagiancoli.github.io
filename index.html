Sobre

Atualmente sou professora do Instituto Federal de São Paulo, no campus de Bragança Paulista, na área de programação e banco de dados. Leciono em componentes curriculares relacionados com Lógica de Programação, Linguagem de Programação C e Python Desenvolvimento Web,

Banco de Dados, Engenharia de Software, Análise Orientada a Objetos. Trabalhei anteriormente em grandes multinacionais com projetos de suporte a sistemas ERP, análises funcionais, desenvolvimento e entregas. Execução de atividades de forma colaborativa e em equipes.

Análise Orientada a Objetos

Diagramas de UML

Neste componente curricular são abordados os conceitos de Orientação a Objetos com aplicação prática na elaboração de Diagramas da Linguagem de Modelagem Unificada (UML).

Ao final do componente curricular os estudantes apresentarão um projeto de um sistema idealizado com a metodologia de UML. As aulas foram atualizadas e elaboradas com o Jupyter Notebook e gerados os arquivos HTMLs.

Download dos materiais de aula.

DWE


Desenvolvimento Web

HTML | CSS | JS
Flask

Neste componente curricular são abordados os conceitos da linguagem de marcação de hipertexto (HTML), formatação de estilos utilizando CSS, e a linguagem de script Javascript.

Desenvolvimento de uma aplicação Flask.

São apresentadas as ferramentas de desenvolvimento atuais de um Desenvolvedor Front-End.

No decorrer das aulas, os estudantes desenvolvem um projeto em grupo utilizando os conceitos adquiridos e o apresentarão no final.

As aulas foram atualizadas e elaboradas com o Jupyter Notebook e gerados os arquivos HTMLs.

Download dos materiais de aula.

Materiais de apoio


LP2 | AOO | DWE

Ferramentas para utilização durante as aulas:

Jupyter Notebook: para criar documentos textos em Markdown, desenvolver códigos em Python e outras linguagens de programação.

Colab Google:é um ambiente de notebooks Jupyter que não requer configuração e é executado na nuvem. Escreva e execute códigos em Python, salve e compartilhe suas análises e acesse poderosos recursos de computação científica, tudo gratuitamente no seu navegador.

Repl.it: é um ambiente de desenvolvimento online compatível com diversas linguagens de programação.

Overleaf: é um editor de LaTeX online fácil de usar. Sem instalação, colaboração em tempo real, controle de versões, centenas de templates LaTeX e mais.

PyCharm: é um ambiente de desenvolvimento integrado usado em programação de computadores, especificamente para a linguagem Python. É desenvolvido pela empresa checa JetBrains.

VsCode: é um editor de código-fonte desenvolvido pela Microsoft para Windows, Linux e macOS. Ele inclui suporte para depuração, controle Git incorporado, realce de sintaxe, complementação inteligente de código, snippets e refatoração de código.

Kahoot: é uma plataforma de aprendizado baseada em jogos, usada como tecnologia educacional em escolas e outras instituições de ensino.

https://www.jetbrains.com/pt-br/pycharm/

Materiais de apoio para os componentes curriculares ministrados por mim no IFSP-BRA. (Informações para utilizar o Git, Markdown).

Documentação do Jupyter Notebook: Como instalar, utilizar etc.
