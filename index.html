Sobre

Atualmente sou professora do Instituto Federal de São Paulo, no campus de Bragança Paulista, na área de programação e banco de dados. Leciono em componentes curriculares relacionados com Lógica de Programação, Linguagem de Programação C e Python Desenvolvimento Web,

Banco de Dados, Engenharia de Software, Análise Orientada a Objetos. Trabalhei anteriormente em grandes multinacionais com projetos de suporte a sistemas ERP, análises funcionais, desenvolvimento e entregas. Execução de atividades de forma colaborativa e em equipes.

LP2


Linguagem de Programação

Introdução ao Python

Nesta disciplina são abordados os conceitos da Linguagem de Programação Python com aplicação em atividades práticas semanais.

No final da disciplina, os estudantes estarão aptos para desenvolver aplicações desde as mais simples até as mais elaboradas.

Além de apresentarem um projeto final desenvolvido em grupo, abrangendo os conteúdos ministrados no decorrer da disciplina.

As aulas foram atualizadas e elaboradas com o Jupyter Notebook e gerados os arquivos HTMLs.

Download dos materiais de aula.

AOO


Análise Orientada a Objetos

Diagramas de UML

Nesta disciplina são abordados os conceitos de Orientação a Objetos com aplicação prática na elaboração de Diagramas da Linguagem de Modelagem Unificada (UML).

Ao final da disciplina os estudantes apresentarão um projeto de um sistema idealizado com a metodologia de UML. As aulas foram atualizadas e elaboradas com o Jupyter Notebook e gerados os arquivos HTMLs.

Download dos materiais de aula.

DWE


Desenvolvimento Web

HTML | CSS | JS
Flask

Nesta disciplina são abordados os conceitos da linguagem de marcação de hipertexto (HTML), formatação de estilos utilizando CSS, e a linguagem de script Javascript.

Desenvolvimento de uma aplicação Flask.

São apresentadas as ferramentas de desenvolvimento atuais de um Desenvolvedor Front-End.

No decorrer das aulas, os estudantes desenvolvem um projeto em grupo utilizando os conceitos adquiridos e o apresentarão no final.

As aulas foram atualizadas e elaboradas com o Jupyter Notebook e gerados os arquivos HTMLs.

Download dos materiais de aula.
